== Description ==

This plugin saves inputs from “[Contact Form 7](https://wordpress.org/plugins/contact-form-7/)” webforms and organize them.
With “**Save Leads**” you will be able to watch and track leads through your E-mail or a designated dashboard.
You can also mark the stages of the treatment and leave comments.

== Features ==

* Saves leads from “[Contact Form 7](https://wordpress.org/plugins/contact-form-7/)”
* Leads display
* Email
* Dashboard
* Mark treatment stages
* Add Comments

== Installation ==

1. Download, install and Activate the plugin “[Contact Form 7](https://wordpress.org/plugins/contact-form-7/)” *(Skip this step if you previously installed it)*
2. Download “**Save Leads**” plugin.
3. Upload the “**Save Leads**” folder to your “/wp-content/plugins/ directory”
4. Activate the plugin through the “Plugins” menu in WordPress
5. Enjoy managing your leads…
